System Bot OpenWRT Monitoring Page

The following monitoring APIs (sort of - they're cgi-bin/ scripts) are available on this host:

  • /cgi-bin/network/interface/dump
  • /cgi-bin/network/interface/
  • /cgi-bin/network/interface?name_of_interface_here
  • /cgi-bin/network/device
  • /cgi-bin/system/board
  • /cgi-bin/system/cpu
  • /cgi-bin/system/info
